## Installation
bash
git clone https://github.com/KKPitsadee44/merge-sorted-arrays.git
cd merge-sorted-arrays
npm install

## Usage
```typescript
import { merge } from './src/merge';

const result = merge([1, 3], [2, 4], [6, 5]);
console.log(result); // 1, 2, 3, 4, 5, 6
```
